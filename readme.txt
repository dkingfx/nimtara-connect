Connect your WordPress site to the Nimtara AI content platform.

== Description ==

Nimtara Connect pairs your WordPress site with the Nimtara AI content
operations platform. Once installed, Nimtara can publish AI-generated
articles directly to your site as drafts, pending posts, or live content.

Features:
* Secure API key authentication
* Maps Nimtara pillars to WordPress categories
* Auto-uploads featured images to your media library
* Stores Nimtara run IDs as post meta for full traceability
* Optional callback webhook so Nimtara knows when posts go live
* Admin dashboard showing all Nimtara submissions

== Installation ==

1. Upload the `nimtara-connect` folder to `/wp-content/plugins/`
2. Activate the plugin via the Plugins menu
3. Go to Settings → Nimtara Connect
4. Copy the Submit Endpoint and API Key into your Nimtara publish connection
